# Юнит-экономика прайса × 4 схемы маркетплейсов — отчёт (черновик)

Прогон текущего прайса поставщика по 4 связкам (WB / Ozon × ФБС / ФБО):
маржа, прибыль на единицу, минимальная наценка для выхода в «зелёную» зону.
Источник расчётов — внутренний калькулятор; цифры на дату генерации,
все параметры (налог, ДРР, скидка закупки, тарифы логистики, комиссии)
перечислены в шапке отчёта.

**Отчёт:** [`index.html`](./index.html) — открой напрямую или через GitHub Pages
этого репозитория (`https://<owner>.github.io/<repo>/`).
