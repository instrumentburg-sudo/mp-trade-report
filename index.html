Юнит-экономика текущего прайса KingQueen × 4 связки

Прогон 2026-05-12: 133 SKU × {WB, Ozon} × {ФБС, ФБО} × {наценки ×1.6, ×2, ×2.5, ×3} = 2128 расчётов. Калькулятор — monitoring/unit_economy.py (модель вкладки «юнит ФБО», 31 колонка).

Параметры и источники

  • Себес = цена прайса KingQueen × (1 − 28%) (партнёрская скидка KQ; KQ_PARTNER_DISCOUNT в .env). Прайс — KINGQUEEN_PRICE_PATH (кеш-час).
  • Цена продажи — развёртка по наценкам к Себесу (×1.6, ×2, ×2.5, ×3); наценка = фактическая цена продажи («Цена со скидкой»). Референс для сводки/таблиц — ×2.
  • Габариты / категория: из вкладки «юнит ФБО» (unit-fbo-sheet-2026-05-11.csv) для 20 SKU, что там есть; для остальных 113 — инференс по названию (флаг лог≈ — логистика приблизительная, точные габариты с карточки поставщика при заведении на МП).
  • Ozon — живые ставки кабинета из monitoring/snapshots/2026-05-12/ozon-prices.json (13/133 SKU прайса нашлись по offer_id): комиссия sales_percent_fbo/fbs, эквайринг, тарифы логистики FBO (магистраль + последняя миля), обработка отправления FBS (first mile), обратка. Нет в кабинете → статичные шкалы compute() (флаг Ozon: нет в кабинете).
  • WB — живые тарифы Seller API из monitoring/snapshots/2026-05-12/wb-tariffs.json (/api/v1/tariffs/box, /api/v1/tariffs/commission, /api/tariffs/v1/acceptance/coefficients): для WB-ФБС — тариф короба (₽ за первый/дополнительный литр), КЛС×КТР склада, КВВ-комиссия по категории; для WB-ФБО — категорийная комиссия для не-вкладочных SKU. Дефолт-склад: Рязань (Тюшевское) (ЦФО, мин. коэф логистики). Флаг WB: live тарифы.
  • WB-ФБО для SKU из вкладки: store (AID Tools / AID Official), «Кэф склада» 150 %, «Выкуп», «Базовый тариф» / «Логистика с кэфом» / «Обратная логистика» — из вкладки (склад/магазин-зависимы; вкладка = golden reference, снапшотом не перебивается). Вне вкладки — аппроксимация базового тарифа (флаг WB-ФБО лог≈) + категорийная комиссия из снапшота, если он есть.
  • WB-ФБС: тариф короба и «Кэф склада» = КЛС×КТР — live из снапшота WB (фолбэк дефолт 1.32 + сетка ресёрча, требует проверки в LK), КГТ+ — из габаритов. Ozon-ФБС: фикс-тариф по объёму 06.04.2026 + обработка отправления. Ozon-ФБО: логистика доставки (live или функция объёма), хранение 0 ₽/ед (размещение ≤120 дн — бесплатный период), приёмка — моно на доверительной ≈0; кластерную наценку нелокальных не закладываем.
  • Налог — ИП Дедков 6.2 % (УСН Д-Р мин 1.2 % + НДС 5 %). ДРР 12% (поправка Тимура). Эквайринг WB 1.3 % / Ozon live (≈1 %, фолбэк 1.8 %). Упаковка 30 ₽. Доставка партии до МП 20 ₽ (ФБС) / 50 ₽ (ФБО). Выкуп дефолт WB 92 % / Ozon 95 % (WB-ФБО — из вкладки 85–90 %). Брак 1.5 %. Ozon: подписка «Управление отзывами» 1 % + переупаковка возвратов 80 ₽.

Вердикт (подход Тимура #3731): 🟢 маржа ≥ 15 % · 🟡 5–15 % (рабочий минимум, после СПП реальная выше) · 🔴 < 15 % при цене ниже пола или < 5 % / убыток. Пол: цена ≥ 1.4 × Себес (ozon/guards.MIN_MARGIN_OVER_COST) — ниже него связка не «зелёная» даже при формально достаточной марже.

Что уточнить в LK перед запуском

  1. Закупка KQ — партнёрская скидка 28 % (проверить актуальность в кабинете KingQueen).
